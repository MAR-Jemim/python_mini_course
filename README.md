## **Python** (mini course)
> This course is designed to introduce **Python** in around
> 1 month (10-12 classes). If you practice consistently, 
> this course will benefit you, inshaAllah. 
> Otherwise, it may end up being a waste of time.  
> 
> To make this useful remember to
> - Practice at least 1 hour every weekday.
> - Practice at least 4-5 hours over the weekend.
> - Visit [W3Schools/Python](https://www.w3schools.com/python/) 
> to study more.
> - Ask your club members whenever you run into difficulties.

### By the end of this course

If you practice consistently throughout the course, inshaAllah, you will be able to:

- Write simple Python programs.
- Display output and take input from users.
- Use variables and different data types.
- Work with strings and lists.
- Make decisions using `if`, `elif`, and `else`.
- Repeat tasks using loops.
- Organize your code using functions.
- Solve basic programming problems using Python.
- Continue learning Python on your own using online resources and documentation.
